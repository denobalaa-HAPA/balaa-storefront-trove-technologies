# Trove Technologies - Electronics & Phone Repair

Nairobi's Premier Tech Store & Repairs - BALLAA Storefront

## Business Information

- **Name:** Trove Technologies
- **Location:** Norwich Union Building, 5th Floor, Suite 4, Nairobi, Kenya
- **Products:** Smartphones, Laptops, TVs, Accessories
- **Services:** Phone Repairs, Screen Replacement, Battery Replacement, Charging Port Repair, Camera Repair, Water Damage Assessment, Software Repair, Speaker & Microphone Repair, Back Glass Replacement

## Features

- Product Catalog with Specifications
- Repair Requests via Messaging
- Category Filtering
- Contact & FAQ Pages

## Run Locally

1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`

## Deployment

This storefront is deployed via Firebase Hosting. See `deploy_storefront.ps1` for deployment orchestration.
